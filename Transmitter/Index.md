# 操作摇杆

在操作摇杆中，共有三种操作：**模式切换**、**方向操纵**和**推力操纵**。

## 模式说明

```{figure} Images/Front.png
:align: center
遥控器说明
```

以下以遥控器为例进行说明，手持地面站摇杆类同。

* **锁定**：锁定后，无法通过摇杆操作载具。
* **遥控器**：通过摇杆操作载具。
* **开始任务**：执行由App发送的任务，从第一个任务点开始执行。

## 操作流程

```{figure} Images/ET10.png
:align: center
遥控器
```

1. **准备**：请将遥控器左摇杆调至垂直中位，并将SB开关置于最上方，以确保遥控器被锁定。
2. **开机**：长按开机键来开启遥控器。开机后，会听到蜂鸣器的提示音，此时点击触控屏上的确定按钮，完成启动确认。
3. **电量**：触控屏右上角显示遥控器的电压值，而左下角Rx则显示接收机的电压值。
4. **操作**：将SB开关拨至中间位置以解锁，之后即可使用摇杆来操控载具。

```{attention}
操作遥控器时，务必注意将微调T1至T4保持在中立位置，即数值应设定在0。只有当遥控器的摇杆出现偏差，影响了控制精度时，才需要微调T1至T4来进行校准。
```
